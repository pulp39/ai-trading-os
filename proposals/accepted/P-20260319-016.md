Proposal P-20260319-016
Title: simulated_order Task Artifact Definition — Bounded Execution Path
Date: 2026-03-19
From: Proposer (Claude)
Type: Execution Implementation Layer — Task Artifact Definition
Constitutional Impact: None
Scope: simulated_order用approved task artifact・bounded execution path定義（最小）

Background
P-20260319-015でsimulated_orderが制度レベルで定義されました。しかし第4回テストで以下が確認されました：
欠けているもの：
  ① simulated_order用 bounded execution pathが未定義
  ② simulated_order用 approved task artifactが未存在

これらは「OpenClawに何をしてよいか」を与える
境界そのものであり、制度化が先行すべき。

Change 1 — approved task artifactの定義
名前と場所：
ファイル名：registrar_queue/simulated_order_task.json
役割：OpenClawが参照するapproved task artifact
最小構造：
json{
  "task_type": "simulated_order",
  "task_version": "1.0",
  "approved_by": "ATOS",
  "approval_date": "2026-03-19",
  "source_proposal": "P-20260319-015",
  "bounded_path": "scripts/openclaw/run_simulated_order.py",
  "allowed_symbols": ["7203", "8306", "6758"],
  "allowed_order_types": ["market"],
  "allowed_sides": ["buy", "sell"],
  "max_quantity": 100,
  "requires_aab": true,
  "requires_dry_run": true,
  "external_order_forbidden": true
}
```

---

### Change 2 — bounded execution pathの定義

**パス：**
```
scripts/openclaw/run_simulated_order.py
```

**実行フロー：**
```
① AAB / TMP受信（Receiver）
② trust_token検証
③ task artifactの存在確認
   → registrar_queue/simulated_order_task.json
④ 注文パラメータ生成（dry_run）
⑤ 外部送信しない（確認）
⑥ execution_recorded + execution_subtype=simulated_order を記録

Change 3 — 注文パラメータの最小集合
simulated_orderで生成してよいパラメータ：
json{
  "symbol": "8306",
  "order_type": "market",
  "side": "buy",
  "quantity": 100,
  "estimated_price": 2750.0,
  "simulated_at": "2026-03-19T10:15:00+09:00",
  "dry_run": true,
  "external_order_sent": false
}
```

**生成してはいけないもの：**
```
❌ 実際の注文ID
❌ 証券会社APIへの接続情報
❌ 実約定価格

Change 4 — execution_recordedフォーマット確定
P-20260319-015で定義した形式を実装レベルで固定します：
json{
  "event_type": "execution_recorded",
  "execution_subtype": "simulated_order",
  "aab_id": "AAB-YYYYMMDD-NNN",
  "target_agent_id": "openclaw",
  "status": "simulated",
  "order_params": {
    "symbol": "8306",
    "order_type": "market",
    "quantity": 100,
    "side": "buy",
    "estimated_price": 2750.0
  },
  "dry_run": true,
  "external_order_sent": false,
  "recorded_by": "OpenClaw/Recorder",
  "recorded_at": "<timestamp>"
}

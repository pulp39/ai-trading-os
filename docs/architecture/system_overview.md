# System Overview

AI Trading OS

Version 1.0  
Date 2026-03-08

---

# System Purpose

The system exists to support cooperative research between humans and artificial intelligence in financial market analysis.

The platform integrates:

Market observation  
AI-assisted hypothesis generation  
Traceable research records  
Controlled trading execution

---

# Institutional Structure

The system operates as a research institution.

Governance  
Defined by the AI Trading Research Council Charter.

Memory  
Maintained by a PostgreSQL database.

Research Activity  
Performed within the VM-A environment.

---

# Architectural Layers

Infrastructure Layer  
Host machine and virtual machines.

Memory Layer  
PostgreSQL research database.

Knowledge Layer  
Git repository containing system documents.

Agent Layer  
Human and AI agents participating in the research council.

Research Layer  
Hypothesis generation, experimentation, and evaluation.

Execution Layer  
Future trading execution system.
